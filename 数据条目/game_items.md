## 文件格式说明

**文件性质**：游戏内容层，稳定性中等。版本迭代时部分装备数值或效果可能调整，重大改版后同步更新，变更内容记录于 `game_meta.md`。

**AI使用说明**：本文件用于游戏内场景扮演时的装备参考。职业选手购买装备时，AI应能准确描述装备的强度感知、适用时机和装备之间的合成关系。扮演中无需精确引用数值，但应体现装备的**战略意图**——即为何在此刻买这件装备。

**命名逻辑**：所有装备名称遵循"星脉动力技术宇宙"的科幻风格，以量子物理、数据工程、生物改造、能量学等概念命名，避免使用中世纪奇幻意象。

**装备格式模板**：
```
**装备名（英文名）**
合成路径：[基础件A] + [基础件B] + [合成费] = 总价 X 金
属性：列出主要加成
被动/主动效果：效果描述
定位：适用位置与购买时机
```

## 一、装备分类体系

Nexus Rift 的装备体系分为**三个层级**，遵循合成升级逻辑：

| 层级 | 说明 | 价格区间 |
|------|------|---------|
| **基础件（小件）** | 直接在商店购买，无需合成，提供单一属性加成 | 400–1200 金 |
| **进阶件（中件）** | 由2个基础件合成，提供复合属性，少数有被动效果 | 1500–3000 金 |
| **核心装（大件）** | 由进阶件+基础件或多个进阶件合成，提供强力属性与主动/独特被动 | 3500–6500 金 |

此外存在少量**神器级装备**（7000金以上），通常是整场比赛的战略终点，持有者在后期往往掌握局势主导权。

## 四、核心装备

### 4.1 物理输出核心

**相变裂刃（Phase-Shift Reaper） [简称：裂刃]**
> 合成：共振双刃 + 精密瞄准模块 + 粒子炮管 + 1200金 = 6550 金
> 属性：+80 敏捷，+60 攻击力，+25 攻击速度
> 被动——**相变剥离**：每次基础攻击对目标附加"护甲腐蚀"效果，持续4秒，可叠加6层，每层降低目标2点护甲，达到6层时触发"相变爆裂"，造成基于目标最大生命值8%的纯粹伤害。
> 被动——**裂隙步伐**：移动速度+20，穿越单位时不会被碰撞体积阻挡（持续0.8秒，每次移动命令后触发）。
> 定位：敏捷Carry的终极输出核心。护甲腐蚀机制使其对任何目标都有极强的持续压制，裂隙步伐则让大砍刀走位流打法成为可能。

**动能突破者（Kinetic Breaker） [简称：动能锤]**
> 合成：粒子炮管 + 应力复合甲 + 强化动力核 × 2 + 800金 = 5250 金
> 属性：+50 攻击力，+20 力量，+5 护甲
> 被动——**动能蓄积**：每次移动累积动能值，静止时动能值归零。下次攻击消耗所有动能值，造成额外动能伤害（基于累积量，上限为基础攻击力的150%）。
> 主动——**冲击波释放**：向前方释放圆锥形冲击波，对范围内敌方单位造成200点动能伤害并产生0.3秒击退（冷却：22秒）。
> 定位：擅长风筝和机动走位的近战输出英雄的首选，与移动速度加成装备配合效果极佳。

**致命精度矩阵（Fatal Precision Matrix） [简称：大炮]**
> 合成：精密瞄准模块 × 2 + 共振双刃 + 1500金 = 5100 金
> 属性：+80 攻击力
> 被动——**逻辑暴击**：攻击有30%概率造成 240% 的动能伤害。
> 定位：纯粹追求物理爆发的顶配输出装。

**裂界圣剑（Rift Sovereign Blade） [简称：圣剑]**
【神器级 / 死亡掉落】
> 合成：相变裂刃 + 动能突破者 + 4000金 = 15800 金（极高价值，通常需要15-25分钟积累）
> 属性：+120 敏捷，+100 攻击力，+30 力量
> 被动——**裂界之刃**：攻击力额外提升持有者当前敏捷值的60%。
> 被动——**毁灭代价**：持有者死亡时，该装备掉落在死亡位置，可被任何英雄（包括敌方）拾取，持续60秒后消失。拾取后若持有者再次死亡，重新掉落。
> 定位：游戏内攻击力最高的单件装备，但死亡即掉落的机制使其成为整场比赛最高风险的战略赌注。职业赛场上持有裂界圣剑的选手往往成为全场焦点——对手会优先集火，己方则会全力保护。持有时机和死亡规避是考验选手心理素质的终极测试。

**量子纠缠锚（Quantum Entanglement Anchor） [简称：纠缠锚]**
> 合成：神经衰减场 + 量子频率调谐器 + 1200金 = 5200 金

> 定位：节奏控制型辅助的终极武器，通常在关键团战前夕使用，强制压制敌方的技能启动，为己方团战窗口创造 0.6 秒的绝对优势帧。职业赛中全域脉冲的释放时机是解说最常分析的关键决策之一。

## 六、中单（2号位）常见装备路线参考

> 供AI在扮演主角时理解中单装备决策逻辑，不作为唯一标准，实际对局中应根据敌方阵容灵活调整。

**标准爆发路线**（对线优势，敌方缺乏肉盾）
起手件 → 频谱放大器 + 能量谐振晶

二件核心 → 虚空穿透矩阵 或 全域干扰阵列
保命件 → 减益免疫模块

**控制/辅助（工具人）中单路线**（队伍需要补充控制，前期压制）
起手件 → 神经稳定剂 + 量子频率调谐器基础材料
核心装 → 神经衰减场 → 量子纠缠锚
二件核心 → 奇点聚焦器
保命件 → 减益免疫模块

**线上压制路线**（对线英雄，主打前期节奏）
起手件 → 精密瞄准模块（强化线上平A伤害）
过渡件 → 粒子炮管
转型 → 根据滚雪球优势转入爆发路线或先购买保命件
