## 文件格式说明

**文件性质**：赛制规则层，稳定性较高。仅在故事涉及晋级、降级、选秀或赛事参与时需要查阅。赛季具体战绩和排名变化记录于 `match_chronicle.md`。

**AI使用说明**：本文件是叙事中涉及赛程安排、积分压力、升降级紧张感、选秀机制的核心参考。职业选手对联赛结构有清晰认知，但未必了解所有细则——教练、经纪人和老将通常比新人更熟悉规则的灰色地带。

## 一、体系总览

Nexus Rift的职业赛事体系由**星脉动力**统一授权和监管，分为四类：

| 赛事类型 | 主办方 | 周期 | 性质 |
|---------|-------|------|------|
| **官方联赛**（S/A/B三级） | 星脉动力 | 赛季制，每赛季一届 | 常规积分赛，决定升降级和世界赛资格 |
| **世界冠军赛** | 星脉动力 | 每赛季末 | 年度最高赛事，全球顶尖战队角逐 |
| **全球季中赛** | 星脉动力 | 每赛季中段 | 各大赛区代表队交流赛，有积分奖励 |
| **挑战者杯** | 第三方（各地区赛事公司） | 不定期 | 邀请赛或资格赛性质，奖金与名誉为主 |

## 二、官方联赛体系

### 2.1 三级联赛结构

```
┌─────────────────────────────────────┐
│         S 级联赛（星脉超级联赛）        │
│  16支战队 · 主客场制 · 全国顶级赛事     │
└──────────────┬──────────────────────┘
               │ 升降级（末位2队降级 / 附加赛1队）
┌──────────────▼──────────────────────┐
│         A 级联赛（星脉挑战联赛）        │
│  20支战队 · 分东西南北四赛区           │
└──────────────┬──────────────────────┘
               │ 升降级（各赛区末位1队降级 / 附加赛1队）
┌──────────────▼──────────────────────┐
│         B 级联赛（星脉发展联赛）        │
│  32支战队 · 各省市区域赛 · 含青训队     │
└─────────────────────────────────────┘
```

### 2.2 S级联赛（星脉超级联赛）

**参赛规模**：16支战队，代表Nexus Rift在中国赛区的最高竞技水平。所有S级战队均需在国内主要联赛城市设立主场，主客场赛制全面执行。

**赛程结构**：
- **常规赛**：双循环赛制（主客场各一次），共30轮，历时约20周
- **季后赛**：常规赛积分前8名晋级，采用BO3单败淘汰赛（前4名有一次败者组机会），决赛为BO5
- **世界赛资格**：S级联赛积分排名前3名获得世界冠军赛参赛名额（第1名直接晋级小组赛，第2、3名参加入围赛）

**积分规则**：
- 胜场：+3分
- 加时赛（第三局）获胜：+2分（败方+1分）
- 负场（两局落败）：0分
- 单场比赛为BO2（两局），双方各赢一局则进入决胜BO1加时赛

> **关于BO2赛制**：S级常规赛采用BO2而非BO3，以控制赛季总时长。这意味着每场比赛理论上有三种结果，"抢到一局平局"在职业圈内被称为"拿到一滴血"，是弱队面对强队时的常见战略目标。

**常规赛主客场安排**：
- 主场比赛在本队主场城市举行，客场比赛赴对方城市出战
- 战队一般以高铁出行，沪江市、北燕市等一线城市战队每赛季客场出行次数约14–15次
- 部分边远城市赛段（如锦官市、渝州市）因距离较远，允许连续排布2–3轮客场以减少出行成本，称为"客场连程"

**S级联赛阵容规则**：
- 每队注册名单最多10名选手（首发5+替补5）
- 单赛季内每名选手上场场次无上限限制，但每场比赛必须在赛前30分钟锁定首发名单
- 外援限制：每队最多注册2名非本国籍选手，单场比赛最多上场1名

---

### 2.3 A级联赛（星脉挑战联赛）

**分区制度**：A级联赛分为东、西、南、北四个赛区，每区5支战队，赛区内先进行单循环积分赛（4轮），赛区前2名进入全国A级总决赛（共8队，采用单败淘汰BO3）。

**战队构成**：
- 部分大俱乐部的"二线战队"（与S级一队不同战队名称，独立运营）在此参赛
- 大量没有大俱乐部背景的独立中小俱乐部占据多数席位
- A级是职业圈的"中间层"——有稳定薪资，但知名度和商业价值远不及S级

**升级通道**：
- A级总决赛冠亚军直接升入S级，取代S级末位两支降级战队
- A级总决赛季军与S级积分倒数第3名进行升降级附加赛（BO5），胜者留/升S级

---

### 2.4 B级联赛（星脉发展联赛）

**规模与性质**：B级是职业体系的基础层，共32支战队，按省市区域划分为8个小赛区，每区4支战队进行单循环积分赛，各区第一名进入全国B级总决赛（共8队）。

**青训队参赛规则**：
- S级俱乐部可在B级联赛注册一支青训队（命名须与一队有明显区分，通常加"青年队"或"Academy"后缀）
- **年龄限制**：青训队名单中所有选手须在赛季开始时未满22周岁，不设最低年龄下限
- **升级限制**：青训队在B级联赛取得升级资格后，**不得以青训队身份晋升A级**。若俱乐部希望将青训体系升至A级参赛，须将该战队独立注册为正式子俱乐部，脱离青训队框架。这一规则被称为"青训墙"，旨在防止大俱乐部用青训队名义垄断A级席位
- 青训队的B级成绩不计入母俱乐部的任何升降级积分，两套体系完全独立运营

**B级的实际功能**：
- 大俱乐部青训梯队的竞训平台
- 地方性小俱乐部的生存赛场
- 星脉草案选秀的重要人才来源地（B级表现突出的选手常被草案关注）
- 没有主客场制要求，通常在城市综合电竞馆或学校体育场举行

### 2.5 升降级完整规则

| 联赛层级 | 降级规则 | 升级规则 |
|---------|---------|---------|
| **S级** | 常规赛末位第15、16名直接降至A级；第14名参加升降级附加赛 | 接收A级上升战队 |
| **A级** | 各赛区最后一名（共4队）直接降至B级；各赛区倒数第2名参加本赛区附加赛 | 总决赛冠亚军升S级；季军参加S级附加赛 |
| **B级** | 各小赛区最后一名（共8队）若属于正式注册俱乐部，降入地区业余联赛（赛季结束后可重新申请B级资格）；若属于青训队，取消参赛资格但不影响母俱乐部 | 总决赛冠军升A级（青训队除外，见青训墙规则） |

> **附加赛说明**：升降级附加赛为BO5，在中立场地举行（通常选择两队主场城市之间的第三城市，或由星脉动力指定场地）。附加赛结果当赛季有效，下赛季重新开始积分。

## 三、星脉草案（选秀机制）

### 3.1 草案概述

**星脉草案（Star Draft）**是由星脉动力主导、每赛季末举办一次的全国性职业选手选拔活动，是非职业选手进入职业圈的主要正式渠道。选秀在赛季总决赛结束后约三周举行，通常在沪江市裂界穹顶旁的星脉会展中心公开直播。

### 3.2 参选资格

以下类别的选手可报名参加当年草案：

| 资格类型 | 说明 |
|---------|------|
| **天梯资格** | 当赛季结束时，天梯排名全服前500名且未持有职业合同的选手，自动获得报名资格 |
| **院校推荐** | 经星脉动力认证的高校电竞联赛（约30所）中，当年表现被院校推荐委员会认定为"全国级别"的选手 |
| **青训推荐** | 俱乐部青训队可向草案委员会推荐旗下未满21岁的选手（即便该选手仍在青训队服役期），推荐名额每俱乐部每届不超过2人 |
| **自由申请** | 持有B级联赛参赛经历的独立选手可提交自由申请，由草案委员会进行资格审核 |

**年龄限制**：参选选手须在草案举办日当天年满16周岁，无上限年龄限制。

### 3.3 选秀顺序与轮次

**选秀顺序原则**：倒序选人，S级联赛当赛季积分排名最低的战队获得第1顺位，依此类推。A级、B级战队亦可参与草案，其选秀顺位排在所有S级战队之后（A级战队按总决赛成绩倒序，B级战队排最后）。

**选秀轮次**：草案共进行**3轮**。每轮所有参与草案的俱乐部依顺序各选1人。俱乐部可以放弃某轮次的选秀权（记为"弃权"）或将本轮顺位转让给其他俱乐部（"顺位交易"，须提前向草案委员会备案）。

**顺位交易**：顺位作为独立资产可以在俱乐部间交易，交易内容通常不对外公开，仅交易结果在草案日当天公示。这是职业圈内部博弈最激烈的幕后操作之一。

### 3.4 草案入选后的安排

- 被草案选中的选手须在草案结束后15日内与选择其的俱乐部签署**新秀合同**，合同年限固定为**2年**（含1次赛季）
- 新秀合同薪资由星脉动力制定统一区间（不得低于S级最低保障薪资，不得超过S级平均薪资的1.5倍），以保护俱乐部对新秀的投资风险
- 若选手在15日内拒绝签约，本届草案资格作废，下赛季可重新参选
- 新秀合同期满后，选手进入自由市场，可自行与任何俱乐部谈判

### 3.5 草案与升降级的交叉逻辑

草案选秀权的特殊机制：

- **降级战队保留优先顺位**：从S级降至A级的战队，在下届草案中依然享有"S级降级队"的顺位优先权（排在所有留守S级战队之后，但在A级战队之前）。这一规则旨在帮助降级俱乐部通过草案补充实力，尽快回升
- **新升S级战队顺位靠后**：从A级新升S级的战队在首届参与的草案中顺位排在所有留守S级战队之后（但优于A级战队），以平衡竞争
- **青训推荐不占用顺位**：青训推荐通道的选手若被选中，不占用该俱乐部的正式选秀轮次，视为"额外签约"，但青训推荐全草案不超过总人数的15%

## 四、世界冠军赛（星脉世界赛）

### 4.1 概述

**星脉世界冠军赛（Rift World Championship，简称RWC）**是每赛季末举办的全球最高赛事。永久主场为沪江市裂界穹顶，固定在每年12月至次年1月举行（跨年赛季调整为联赛结束后约6周）。

### 4.2 参赛名额分配

世界赛名额按全球赛区划分：

| 赛区 | 名额数 | 说明 |
|------|--------|------|
| **中国赛区**（联赛主赛区） | 4席 | 1席直接入组，3席参加入围赛 |
| **韩国赛区** | 3席 | 传统强区，2席直接入组，1席入围赛 |
| **东南亚赛区** | 2席 | 近年快速崛起，全部入围赛 |
| **欧洲赛区** | 2席 | 1席直接入组，1席入围赛 |
| **北美赛区** | 2席 | 1席直接入组，1席入围赛 |
| **其他赛区** | 3席 | 拉丁美洲、中东、独立赛区各1席，全部入围赛 |

**合计**：16支战队进入正赛（其中6席直接入小组赛，10席参加入围淘汰赛争夺剩余10个小组赛名额）。

### 4.3 赛制结构

```
入围赛（10队单败淘汰，BO3）
        ↓ 胜者共5队
小组赛（16队瑞士轮五轮三胜晋级，每轮全部BO3）
        ↓ 共8队
八强赛（BO5，双败制）
        ↓ 共4队
半决赛（BO5，双败）
        ↓ 共2队
总决赛（BO5，在裂界穹顶举行）
```

**总决赛日**是整个电竞圈的年度顶点，裂界穹顶约两万名现场观众，VR同时在线观看人数历届均破千万，高峰赛季曾突破三千万。

## 五、全球季中赛（Rift Mid-Season Clash）

**举办时间**：每赛季联赛常规赛进行至约三分之二时，利用联赛赛程间隔举行，历时约2周。

**参赛资格**：各赛区当时的联赛积分榜首位战队代表赛区出赛。若首位战队因赛程冲突无法参加，顺延至积分第二名。

**赛制**：8支战队，分2组单循环积分，每组前2名进入单败淘汰半决赛，决赛为BO5。

**积分奖励**：季中赛获得前4名的战队，在本赛区的世界赛资格积分中获得额外加权，但不影响联赛积分。

**地位与意义**：季中赛在业界被称为"小世界赛"，胜负直接影响赛区在世界赛名额分配的话语权（历史上若某赛区连续两届季中赛无缘决赛，其直接入组名额有被削减的先例）。对于个别战队而言，季中赛的曝光度有时甚至超过本赛区联赛，是快速刷新国际知名度的捷径。

## 六、挑战者杯（第三方赛事）

**挑战者杯**并非单一固定赛事，而是星脉动力授权给第三方赛事公司举办的系列邀请赛或资格赛的统称。常见形式包括：

- **城市邀请赛**：由联赛城市的地方政府或商业机构赞助，邀请4–8支战队进行城市主题赛事，兼具旅游推广和电竞宣传功能
- **品牌联名赛**：由赞助商举办，通常规模较小（6–8队），奖金池来自品牌方，有时有限定皮肤等游戏内奖励
- **赛区开放挑战赛**：允许A级战队参与的混合邀请赛，是A级战队在联赛之外唯一可能与S级战队同场竞技的机会

**参与与否**：俱乐部对挑战者杯的参与享有完全自主权，星脉动力不强制要求参加，但会提供参与激励（流量曝光资源、官方媒体报道配合等）。S级战队通常选择性参与——在赛季前期和赛季间歇期接受邀请，在联赛冲刺阶段拒绝以保证训练强度。

**规则约束**：挑战者杯须遵守星脉动力的统一规则框架（相同英雄池、相同版本），第三方可自定奖金结构和赛程安排，但不得修改任何游戏内规则。
