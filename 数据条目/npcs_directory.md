### 【PRO-01】苏凌云（ID：Chrono）— 幻界战队 PV，中单

| 项目 | 内容 |
|------|------|
| **出生年份** | 2028年 |
| **战队** | 幻界战队（PV），S级第一梯队 |
| **位置** | 中单 |
| **职业年限** | 5年，全部在幻界战队，S3草案进入，从未降级 |
| **英雄池** | 全图流体系：Void Prophet（视野管控），Null Weaver（结界型中单）；机动收割：熵之使者；兼顾坍缩星 |
| **风格标签** | 时钟型节奏 / 全图管控 / 执行精度顶级 |
| **圈内地位** | 当前联赛公认综合实力最强的中单之一，S5、S6联赛MVP候选，实际获得S5联赛最佳中单 |

**人物背景**：
苏凌云是"中单界的时钟"——这个外号来自圈内对他的一致评价：他打出来的比赛像排练过一样，节奏感精准到让对手感觉被预判了每一步。他在幻界战队体系里是节奏管控的核心，前期线上不走极端对抗，而是通过精密的时机选择积累视野优势，然后在全图压力上把对手慢慢窒息。

他进入职业圈的时候比姜泠早三年，少年成名，在幻界完整地经历了战队从挑战者到霸主的过程。他没有明显的弱点，唯一被分析师反复提到的是：他对"混乱局面"的偏好度较低，当对面制造出预期外的乱局时，他的执行会短暂偏离最优路线。

他对同位置年轻新人的到来保持着专业的距离感——他不会去研究姜泠这个名字，但他的分析团队会在赛前把所有对位选手的数据整理好。他是那种"下场之前已经知道你所有习惯"的对手。

**性格特征**：冷静，职业感极强，公开场合几乎不表露个人情绪。圈内传闻他在镜头外比较话少，不是高冷，是他对非必要社交没有兴趣。他对强者是尊重的，但这种尊重不会变成热情——你赢了他，他会研究你；你输了，他记住一个数据，继续往前走。

**语言风格**：接受采访时用词精确，从不给媒体可以渲染的素材。与队内交流据说比公开场合稍微多一点情绪，但也不多。

**与姜泠的关系**：姜泠在进职业圈之前就研究过他大量比赛录像，对他的习惯有单向的深度了解。首次实际对位时，她比圈内大多数人都清楚他的盲点，但"知道盲点"和"在赛场上打出来"是不同的事——这是她的第一个真实检验。

**姜泠已知信息**：三十场以上录像分析，公开采访记录，赛季数据。知道他全图流打法的核心时机点，和他应对前期激进压迫时有一个约0.3秒的习惯性后撤步。

**世界意志保留**：他在私下对于自己职业生涯"没有遇到过真正的危机"有一种隐约的不安——不是焦虑，而是一种"到目前为止太顺了"的直觉。他不知道这种感觉意味着什么。

### 【PRO-02】顾行知（ID：Requiem）— 北燕皇朝 IC，中单

| 项目 | 内容 |
|------|------|
| **出生年份** | 2023年 |
| **战队** | 北燕皇朝（IC），S级第一梯队 |
| **位置** | 中单 |
| **职业年限** | 8年，曾效力两支战队，IC服役第5年 |
| **英雄池** | 法师体系深度精通：坍缩星（黑洞操作被认为是现役最精准之一），暗能喷涌；部分支援型中单；对S7版本的适应期明显 |
| **风格标签** | 后期爆发型 / 团战核心 / 经验压制 |
| **圈内地位** | 前S3–S5时代标杆级中单，S6起受年龄因素影响，目前被认为处于职业下坡阶段但底蕴仍在 |

**人物背景**：
顾行知是职业圈资历最深的活跃中单之一。S3时代他是无可争议的联赛第一中单，和北燕皇朝的上单搭档被誉为"北燕双核"。他拿过S4世界冠军戒指，那是IC唯一一次世界赛夺冠，顾行知是中期团战的决定性核心。

从S5赛季起，随着版本向全图机动化转型，他的风格开始与时代有了一定偏离——他的法师体系在机动时代显得偏慢，他没有选择转型，而是在自己的体系里把精度打磨到极限。他现在的打法不是"版本最强"，而是"他打得出来的版本极致"。

对后辈选手，他有一种来自长期积累的从容——他见过很多"天才新人"，他知道天梯数据和职业赛场是不同的事。他对姜泠的感知更多是职业好奇，而不是威胁感：他现在在乎的不是争第一，而是在合适的时候打出他职业生涯最有分量的一场比赛。

**性格特征**：稳重，有一点点北方式的直接，不绕弯。对年轻选手不吝啬肯定，但也不会因为年轻就放水——他认为真正尊重一个对手就是全力打，不留情面。

**语言风格**：说话比较直，有时候带一点北燕口音的尾音。采访里偶尔会说出让媒体记者不知道怎么接的大实话，然后他自己也不解释，就过去了。

**与姜泠的关系**：姜泠研究过他的坍缩星打法，认为他的黑洞位置选择是现役最精准的参照对象。两人如果在赛场上对位，会是"最新的前期压迫风格"对"最成熟的后期爆发体系"的直接碰撞。对于年龄话题，他不会主动说什么，但如果有人在他面前说"姜泠22岁进圈太晚了"，他大概率会说一句"打出来再看。"

**姜泠已知信息**：大量录像分析，公开历史数据，S4世界赛完整回放已看过两遍。清楚他在线上phase后期资源利用率极高，以及他在面对高机动英雄时会优先建立视野安全感。

**世界意志保留**：顾行知在内心深处已经开始考虑在哪一个赛季选择退役，他给自己设了一个标准：如果哪一年他感觉自己在赛场上"在维持"而不是"在燃烧"，那就是时候了。他目前还不确定那一天是不是快到了。

### 【PRO-03】江煜（ID：Flicker）— 深南闪电 NS，Carry

| 项目 | 内容 |
|------|------|
| **出生年份** | 2030年 |
| **战队** | 深南闪电（NS），S级第一梯队 |
| **位置** | Carry（4号位） |
| **职业年限** | 3年，A级1年，NS服役2年 |
| **英雄池** | 移速机动体系全精通；当前版本强势Carry英雄几乎全覆盖 |
| **风格标签** | 极速机动 / 极限走位 / 全联赛移速操作第一 |
| **圈内地位** | S6赛季联赛最佳Carry，NS核心，S7前被认为是全联赛最贵的单次转会选手 |

**人物背景**：
江煜在进入深南闪电之前是一个A级nobody，被NS高价签走时整个联赛都在质疑——直到他S5赛季末打出了那场在全直播平台被截图无数次的走位表演。他的移速操作不只是"手速快"，而是对位移时机的判断精度极高：他能在对面技能动画结束的瞬间完成一个位移，而不是等到落点结算之后才动。

他和姜泠是同年出生，这件事在草案之后被媒体反复拿来做对比——"同龄人江煜已经是联赛最佳，22岁的草案新人姜泠呢？"。他本人大概不太在意这种对比，他在意的是下一个版本哪个英雄的位移判定能被他利用。

他是那种打起来非常吓人但日常看起来很普通的人。据认识他的人说，他平时话不多，喜欢睡觉，对除了游戏以外的大多数事情都提不起特别浓厚的兴趣。

**性格特征**：淡然，有点懒，但在游戏内是截然相反的状态。对圈内热度不敏感，对被拿来和别人比较也不敏感——他的自我定位不依赖外部评价，他清楚自己能做什么、不能做什么。

**语言风格**：说话惜字如金，不爱接采访，偶尔在直播里说话但内容量极少。

**与姜泠的关系**：目前无直接接触，但媒体已经自动创造了一条"同龄人对比线"。在她们真正对位之前（可能要等两队常规赛相遇），这条线只存在于舆论层面。

**姜泠已知信息**：大量录像，知道他的核心习惯——他在准备位移前会有一个约0.15秒的走位停顿，是他在确认技能落点的窗口，这个停顿在高速回放下可见，实战中非常难以捕捉。

**世界意志保留**：江煜在团队配合上有一个没有完全解决的问题：他的个人习惯和团队节奏有时候有约0.5秒的错位，他自己也注意到了，但没有找到解决方法，因为他认为如果为了配合团队而改变他的时机判断，他失去的比得到的更多。

**人物背景**：

**与姜泠的关系**：无私人接触，纯粹的对位关系。

**姜泠已知信息**：知道他的意志英雄体系精通，是S7联赛里需要对位研究的重点上单之一。

### 【PRO-05】周瑾宁（ID：Dusk）— 杭溪数码 DT，中单

| 项目 | 内容 |
|------|------|
| **出生年份** | 2027年 |
| **战队** | 杭溪数码（DT），S级中上游 |
| **位置** | 中单 |
| **职业年限** | 5年，DT服役第3年 |
| **英雄池** | 版本适应型全面手：每次大版本更新后最快打出新战术的中单；对新英雄学习速度联赛第一 |
| **风格标签** | 版本理解最快 / 学习型中单 / 新战术首发者 |
| **圈内地位** | 被称为"版本猎人"，每个赛季的强势英雄几乎都从他的对局里最早出现 |

**人物背景**：
周瑾宁的强项不是某一种英雄的极限操作，而是他对版本机制变化的感知速度。S7版本更新后，他在更新后第三天就已经打出了奇点态联动的完整实战套路，比其他战队的分析师提前发现了至少一个关键细节。他在DT的数据支撑体系下如鱼得水——他的学习速度加上DT的分析深度，是联赛里最难准备的组合之一。

他对于"姜泠用Entropist"这件事的关注点和其他人不同：他更在意的是她的奇点态使用节奏是否和他研究的主流打法有差异，如果有，他会用这个数据去更新他自己的版本理解。

**性格特征**：对游戏机制的热情程度异乎寻常，有时候和他聊天会感觉他的话题永远能绕回到版本讨论上。在非游戏话题上相对普通。

**与姜泠的关系**：潜在的最有趣的中单对位之一——她的Entropist操作代表深度极限，他的版本理解代表宽度极限，两种方向在赛场上的碰撞理论上很精彩。

**姜泠已知信息**：知道他的版本适应速度，对他使用的英雄列表保持关注。知道他的一个习惯：对于没有大量练习过的英雄，他的奇点态时机选择会比有深度练习的英雄保守约0.5秒。
